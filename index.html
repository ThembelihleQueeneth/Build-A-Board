Motherboard View

ROM

ROM stands for Read-Only Memory. It's a type of computer memory that stores permanent data and instructions.

Power Unit

A power unit provides energy to operate machinery or equipment, acting as the source of power.

Storage Device

A storage device stores digital data, allowing saving, retrieving, and transferring information.

CPU

The CPU is the brain of the computer, running its operating system and applications.

Ram

RAM provides temporary storage for data the processor is actively using, allowing fast access and multitasking.

CMOS

CMOS stores BIOS settings and system clock information, powered by a small battery on the motherboard.
